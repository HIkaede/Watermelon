# 合成大西瓜



## 模式选择

随缘Play：所有水果随机出现，包括大西瓜

圣雄肝帝：刷出的都是小葡萄

原汁原味：原版，随机掉落的是前5种水果。

作弊模式：刷出的都是西红柿

去除小瓜：刷出橘子到菠萝

暴力吃瓜：刷出西瓜

## 分数选择

两倍暴击：原有基础上翻2倍

五倍暴击：原有基础上翻5倍

十倍暴击：原有基础上翻10倍

百倍暴击：原有基础上翻100倍

千倍暴击：原有基础上翻1000倍

万倍暴击：原有基础上翻10000倍

十万伏特：原有基础上翻100000倍
